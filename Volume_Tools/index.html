輸出結果

可選 1個SIZE一行 / 3個SIZE一行 / 全部一行
輸出規則

1. DIM 後面的數字 = 箱數
2. 包裹個數會另外統計到 TOTAL 包裹個數
3. 可選輸出排版：
• 1 個 SIZE 一行
• 3 個 SIZE 一行
• 全部 SIZE 一行
4. VOL 取整方式可選：
• ROUND（四捨五入）
• ROUNDUP（無條件進位）
• ROUNDDOWN（無條件捨去）
5. VOL 小數位可選：0 / 1 / 2
